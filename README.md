Deskripsi Aplikasi

Medan Pocket adalah aplikasi mobile yang dibangun menggunakan bahasa pemrograman Dart yang bertujuan untuk membantu wisatawan mengetahui informasi mengenai lokasi wisata di kota Medan dengan mudah.
